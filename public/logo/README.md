# mena climate — logo files

Wordmark is **mena climate** (two words). Typeface: **Space Grotesk, weight 600**, letter-spacing
−0.02em. The "dawn dot" is a circle with a 135° gradient from living green `#43C892` (top-left) to
desert gold `#D8B262` (bottom-right). Clear space: keep at least one dot-height clear on all sides.

## Files

| File | What it is | Use |
|---|---|---|
| `mena-climate-logo-light.png` | Light wordmark, transparent bg (1852×266) | Place on dark / navy backgrounds |
| `mena-climate-logo-dark.png` | Dark wordmark, transparent bg | Place on white / light backgrounds |
| `mena-climate-mark.png` | Dot mark only, transparent bg | App icon / favicon / avatar |
| `mena-climate-logo.svg` | **Editable vector**, light wordmark | Scalable — web, Figma, print |
| `mena-climate-logo-dark.svg` | Editable vector, dark wordmark | For light backgrounds |
| `mena-climate-mark.svg` | Editable vector, dot only | Fully self-contained vector |

## Important — fonts in the SVG wordmark files
The `*-logo*.svg` files use **live text** in Space Grotesk (pulled from Google Fonts), so they render
correctly in any browser and in Figma (when online). In **Adobe Illustrator** the font must be installed
locally (it's free: https://fonts.google.com/specimen/Space+Grotesk) — then open the SVG and run
**Type → Create Outlines** to lock the wordmark to vector paths for fully portable, font-independent
artwork. The `mena-climate-mark.svg` (dot only) has no text and is already fully portable.

The PNGs are pre-rendered with the correct font and need no font installed.
